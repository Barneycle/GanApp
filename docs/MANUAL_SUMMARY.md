# User Manuals Summary

## Overview

This directory contains comprehensive user manuals for both GanApp Web and Mobile applications. The manuals are designed to help users of all roles (Participants, Organizers, and Administrators) effectively use the GanApp platform.

## Files Created

### 1. WEB_USER_MANUAL.md
- **Size:** Comprehensive guide covering all web features
- **Sections:** 8 main sections with detailed subsections
- **Audience:** Participants and Organizers
- **Content:**
  - Introduction and system requirements
  - Getting started guide
  - Participant features (events, certificates, surveys, photos)
  - Organizer features (event creation, management, certificates, QR codes)
  - Troubleshooting guide
  - FAQ section
  - Glossary

### 2. MOBILE_APP_USER_MANUAL.md
- **Size:** Comprehensive guide covering all mobile app features
- **Sections:** 8 main sections with detailed subsections
- **Audience:** Participants and Organizers
- **Content:**
  - Introduction and system requirements (iOS/Android)
  - Getting started and installation
  - Participant features (optimized for mobile)
  - Organizer features (QR scanner focus)
  - Troubleshooting guide
  - FAQ section
  - Glossary

### 3. ADMIN_USER_MANUAL.md
- **Size:** Comprehensive guide covering all administrative features
- **Sections:** 16 main sections with detailed subsections
- **Audience:** System Administrators Only
- **Content:**
  - Introduction and system requirements
  - Admin dashboard overview
  - User management (create, edit, ban, archive, role changes)
  - Event management (system-wide)
  - Cancellation requests processing
  - Analytics and reports
  - System settings (maintenance mode, security)
  - Notification management
  - Database maintenance
  - Activity logs and audit trails
  - Organization management

  - Glossary

### 4. README.md
- **Purpose:** Overview and quick reference
- **Content:** File descriptions and quick PDF generation options

## Manual Structure

Both manuals follow the same structure:

1. **Introduction** - Overview, user roles, and platform capabilities
2. **System Requirements** - Minimum and recommended requirements
3. **Getting Started** - Account creation, sign in, profile setup, navigation
4. **For Participants** - Complete feature guide
5. **For Organizers** - Event management guide
6. **For Administrators** - System administration guide
7. **Troubleshooting** - Common issues and solutions
8. **FAQ** - Frequently asked questions
9. **Glossary** - Key terms and definitions

## PDF Generation

PDF versions can be generated using multiple methods:

1. **Pandoc** (Recommended) - High-quality PDFs with LaTeX
2. **VS Code Extension** - Easy conversion within editor
3. **Online Converters** - Quick conversion without installation
4. **GitHub/GitLab** - Browser print to PDF
5. **Python Tools** - Programmatic conversion
6. **Docker** - Containerized conversion

See `PDF_GENERATION_GUIDE.md` for detailed instructions.
